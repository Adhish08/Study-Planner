How to get your Canvas token
⚠ Treat your token like a password. Never share it. Set an expiry date for extra safety.
1
Log in to canvas.anu.edu.au
2
Click your Account icon (top-left) → Settings
3
Scroll to Approved Integrations → click + New Access Token
4
Set Purpose: Study Planner and set an expiry date (e.g. end of semester)
5
Click Generate Token and copy the result
6
Paste it into the token field in this app. It only lives in memory — never saved to disk.
ℹ This app only reads the data you explicitly allow in the permissions dialog.
